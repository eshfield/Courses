# Courses

### Приложение для просмотра курсов

Тестовое задание

## Функции

- загрузка курсов по API
- возможность добавить курс в избранное
- экран с избранными курсами (хранятся локально)
- поддержка Pull-to-Refresh

## Технологии

- Coil
- Compose
- DataStore Preferences
- Haze
- Koin
- Navigation
- Retrofit

Схема ответа от сервера с тестовыми данными находится в файле `courses_mock.json`. Эти же данные отдаются от расположенного в интернете временного API на сервисе Beceptor

Так как тестовый API не поддерживает приём запросов на добавление курса в избранное, то из первоначальной схемы было убрано поле `hasLike` — вся логика по работе с избранными курсами производится локально
